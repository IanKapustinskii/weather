# Приложение Погоды

Адаптивное приложение для отображения погоды с поиском городов и настройкой единиц измерения для температуры, осадков, ветра и времени.

## Возможности

- **Поиск города**: Найдите погоду для любого города в мире.
- **Переключение единиц**: Смена между метрической и имперской системами:
  - Температура (°C / °F)
  - Осадки (мм / дюймы)
  - Скорость ветра (м/с / mph)
  - Формат времени (24ч / 12ч)
- **Настройка единиц отдельно**: Можно задать единицы для каждого параметра отдельно.
- **Адаптивный дизайн**: Работает на десктопе, планшете и мобильных устройствах.
- **Данные в реальном времени**: Получение информации о погоде с помощью:
  - API [Open-Meteo](https://open-meteo.com/) для прогнозов
  - Геокодинг для поиска городов
- **Интерактивный интерфейс** на React для быстрого и плавного обновления.

## Используемые технологии

- React + TypeScript  
- Vite для сборки и разработки  
- Open-Meteo API  
- CSS для стилизации и адаптивного дизайна  

## Использование

### Поиск города

1. Введите название города в поле поиска.
2. Выберите нужный город из предложений.
3. Текущая погода и прогноз обновятся автоматически.

### Переключение единиц

- Глобальное переключение: меняет все измерения сразу между метрическими и имперскими единицами.
- Индивидуальные единицы: настройте температуру, осадки, ветер и время отдельно.

### Просмотр погоды

- Текущая температура, осадки, скорость ветра и время
- Прогноз на день и на часы
- Динамическое обновление на основе выбранных единиц и местоположения
